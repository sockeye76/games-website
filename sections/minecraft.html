Reminiscence

"Reminiscence is a modpack which aims to combine the gameplay, aesthetics, and general design philosophies of Alpha and Beta versions of Minecraft with the convenience and creative options provided in modern Release versions but with a twist in the form of content mods which add to the experience."

In order to play, please download and install the latest version of Prism Launcher.
